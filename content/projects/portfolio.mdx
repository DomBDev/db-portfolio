# Overview

A modern developer portfolio built with Next.js 15.5 and TypeScript, featuring MDX-driven project showcases, responsive design, and a clean, accessible interface. The site combines modern development practices with performance optimization.

## Key Features

- **Content Management**
  - MDX-based project documentation with type-safe frontmatter using Zod
  - Dynamic project routing with optimized image loading
  - Responsive image handling through Next/Image component

- **Technical Stack**
  - Next.js 15.5 App Router with server components
  - Type-safe development with TypeScript and Zod validation
  - Tailwind CSS with custom theming system (dark/light modes)
  - Vitest for unit testing critical components

- **User Experience**
  - Responsive design with mobile-first approach
  - Dark/light theme with custom color palette
  - Optimized image loading and performance
  - Secure contact form with email integration

## Implementation Details

- Custom theme system using CSS variables and Tailwind
- Server-side MDX processing for optimal performance
- Type-safe email handling with nodemailer
- Comprehensive test coverage for core functionality
